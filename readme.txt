位置合わせプログラム_修正

修正版では5つのファイルから構成されています。
(objLoader.py、objWriter.py、ICP.py、positionfit.py、startprogram.py)
これらのファイルはすべて同じフォルダに保存しておいてください。

起動するファイルはstartprogram.pyです。今までとは違うのでご注意ください。

今までは位置合わせプログラムと同じフォルダに3Dモデルのファイルを入れていましたが、
修正版ではその必要はありません。プログラム内でエクスプローラーを起動して選択するようにしました。

また、新しい外部モジュール(wxPython)を使ったのでインストールをお願いします。
Anaconda Promptで
pip install wxPython
と打ち込んで実行をお願いします。

入力の仕方は、「**モデル入力」ボタンを押すとエクスプローラーが起動するので入力するモデルを選択してください。
「位置合わせ実行」ボタンを押すと再びエクスプローラーが起動するので保存するファイル名を入力してください。
すると、位置合わせが自動で実行されます。

大きさを手動で入力したい場合はチェックボックスにチェックを入れて値を入力してください。


わからないことやエラーがあった場合はSlackやメールなどで聞いてください。
